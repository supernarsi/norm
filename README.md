# norm

## how to use:

### 1. create a config file

create a `.norm-db.env` file under your root path, and edit the file content like `.norm-db.env.example`

```dotenv
[DATABASE]
HOSTNAME =
DATABASE =
USERNAME =
PASSWORD =
HOSTPORT = 3306
```

### 2. exec

goto vendor dir, and run

```shell
./bin/norm-create
```
